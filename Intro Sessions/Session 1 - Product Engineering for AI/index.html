# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

ollama pull qwen2:7b

ollama run qwen2:7b

Step 1: Setting Up EC2

  1. Log in to AWS Console and navigate to EC2
  2. Launch a new instance (t2.large recommended)
  3. Select Ubuntu 22.04 LTS as your AMI
  4. Configure security group to allow SSH access
  5. Create or use an existing key pair
  6. Launch instance and connect via SSH

ssh -i your-key.pem ubuntu@your-instance-public-ip

Step 2: Installing Ollama


# Update system packages
sudo apt update && sudo apt upgrade -y

# Install dependencies
sudo apt install -y curl

# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Verify installation
ollama --version

Ollama provides a simple API and CLI for running LLMs locally

Step 3: Deploying Qwen2.5


# Pull the Qwen2.5 model (7B parameter version)
ollama pull qwen2:7b

# This will download ~4GB of data, be patient!

About Qwen2.5

  • Developed by Alibaba Cloud
  • 7B parameters (compact yet powerful)
  • Good performance-to-size ratio
  • Supports multiple languages

Step 4: Interacting with the Model


# Start a chat session with the model
ollama run qwen2:7b

# Try these prompts:
> Explain the concept of transfer learning in AI
> Write a short poem about Tulsa
> How would you implement a recommendation system?

You can also use the Ollama API to integrate with applications:


# Start the Ollama server
ollama serve

# In another terminal, use curl to query the API
curl -X POST http://localhost:11434/api/generate -d '{
  "model": "qwen2:7b",
  "prompt": "What is machine learning?"
}'

AI Model Terminology

Term Definition
LLM Large Language Model, a type of AI model trained on vast text data
Inference Process of using a trained model to make predictions
Token Basic unit of text processing in LLMs (word parts or characters)
Quantization Process of reducing model precision to improve performance
Context Window The amount of text a model can consider at once
Fine-tuning Adapting a pre-trained model for specific tasks
